Efficient Portfolio Optimization Using MPT

Pandas, Numpy, Matplotlib, SciPy, Yahoo Finance


This project applies Markowitz Portfolio Theory to construct optimal domestic and
international investment portfolios. The domestic minimum variance portfolio achieved 11.67% return,
while the international minimum variance portfolio yielded 19.42% return. The tangency portfolios provided higher
returns (55.45% domestic, 52.93% international), outperforming NIFTY 50 in growth potential.

Sentiment Analysis of Financial News

Pandas, Numpy, Matplotlib, Seaborn, Scikit-learn, WordCloud


Developed a sentiment analysis model for economic headlines using TF-IDF Vectorization
and Logistic Regression, later enhanced with an Artificial Neural Network (ANN). Achieved an AUC score of
91.7%,and accuracy of 83.5%. Evaluated model performance using confusion matrix and predict_proba,
transforming a multi-factor model into binary classification for better interpretability.
